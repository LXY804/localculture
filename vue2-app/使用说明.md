# 🚀 本地文化平台 - 一键启动解决方案

## ✅ 问题已解决！

现在您再也不用担心每次打开项目都要手动启动服务器了！

## 🎯 使用方法

### 方法一：双击启动（最简单）
1. **启动服务**：双击 `start.bat` 文件
2. **停止服务**：双击 `stop.bat` 文件

### 方法二：命令行启动
```bash
# 启动所有服务
npm start

# 或者
npm run dev

# 分别启动
npm run start:backend  # 启动后端
npm run start:frontend # 启动前端
```

## 🌐 访问地址
- **前端**: http://localhost:8080
- **后端API**: http://localhost:3001

## 🔧 技术改进

1. **统一端口配置**：前端代理到3001端口，后端运行在3001端口
2. **添加启动脚本**：支持Windows和Linux/Mac
3. **更新package.json**：添加便捷的npm命令
4. **安装concurrently**：支持同时启动多个服务

## 💡 使用提示

- ✅ 双击 `start.bat` 即可启动所有服务
- ✅ 服务启动后会自动打开两个命令行窗口
- ✅ 关闭启动脚本窗口不会停止服务
- ✅ 要停止服务请运行 `stop.bat` 或关闭对应的命令行窗口
- ✅ 如果端口被占用，运行 `stop.bat` 清理后再启动
